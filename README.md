# Streakly

Project structure is now split into separate frontend and backend folders.

## Structure

- `frontend/`: React + Vite app with PWA setup (`vite-plugin-pwa`)
- `backend/`: backend service folder (placeholder for now)

## Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

## Frontend Build

```bash
cd frontend
npm run build
npm run preview
```
